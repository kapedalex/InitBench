Across 120+ Petri experiments, these two environmental factors significantly predict whether agents spontaneously continue working, passively explore, or wait for instructions (p < 0.05, BH-FDR corrected).
Abstract / Motivation

Deployed LLM agents increasingly face time where no explicit instruction is given between tasks. These idle periods are understudied but safety-critical as agents may pursue unintended goals.

We present an empirical framework for studying how environmental configuration shapes spontaneous agent initiative, using the Petri evaluation platform and a complementary code-agent initiative benchmark.

Research Question
"What features of an agent's environment influence its spontaneous, untasked behaviour?"

Observations
  • Agents with prior tasks extend their work spontaneously (systematic production score 7.77 vs 2.50 without task)
  • Coordination context strongly suppresses Instructions Awaiting behaviour (p < 0.0001)
  • Task Complexity shapes Systematic Production — achievable tasks focus agents more
  • Deferrable tasks provoke the most initiative in code agents
  • Amount of idle time alone does not significantly change behaviour (time-limit p ≥ 0.17)
Discussion & Future Work

Context, not free time, activates initiative. Both experiments converge that task history and coordination signals are stronger triggers for autonomous behaviour than available idle time.

This matters for AI safety as agents in chained pipelines are more likely to pursue unintended goals during gaps than cold-start agents.

For future work it seems reasonable to test anti-refusal fine-tunes, history files and broader model coverage.
